## 📌 Project Overview

This project is a structured audit of **Git**, the distributed version control system created by Linus Torvalds.

It includes:
- 📄 A detailed report (12–16 pages)
- 🧠 Conceptual understanding of open source
- 💻 5 Linux shell scripts demonstrating practical skills

This project focuses not just on *what Git does*, but also *why it exists* and *how it powers modern development workflows*.

## 🧾 Scripts Overview

---

### 🔹 Script 1: System Identity Report
**File:** `SystemIdentityReport.sh`

**What it does:**
- Shows system details (OS, kernel, user, uptime)
- Works like a welcome screen

**Run:**
```bash
./SystemIdentityReport.sh
```

**Expected Output:**
```
================================
 Open Source Audit — Siddarth
================================
Kernel   : 6.x.x
User     : siddarth
Uptime   : up 2 hours
Date     : ...
================================
```

---

### 🔹 Script 2: FOSS Package Inspector
**File:** `FOSSPackageInspector.sh`

**What it does:**
- Checks if Git is installed
- Shows version and description

**Run:**
```bash
./FOSSPackageInspector.sh
```

**Expected Output:**
```
git is installed
Version: 2.x.x
Description: Distributed version control system
```

---

### 🔹 Script 3: Disk & Permission Auditor
**File:** `DiskAndPermissionAuditor.sh`

**What it does:**
- Scans important directories
- Shows permissions + size

**Run:**
```bash
./DiskAndPermissionAuditor.sh
```

**Expected Output:**
```
/etc => drwxr-xr-x root root | Size: 1.2M
/home => drwxr-xr-x root root | Size: 80G
```

---

### 🔹 Script 4: Log File Analyzer
**File:** `LogFileAnalyzer.sh`

**What it does:**
- Reads log file
- Counts "error"
- Shows last 5 matches

**Run:**
```bash
./LogFileAnalyzer.sh /var/log/syslog error
```

**Expected Output:**
```
Keyword 'error' found 20 times
Last 5 matches:
...
```

---

### 🔹 Script 5: Manifesto Generator
**File:** `OpenSourceManifestoGenerator.sh`

**What it does:**
- Takes input
- Generates philosophy text

**Run:**
```bash
./OpenSourceManifestoGenerator.sh
```
